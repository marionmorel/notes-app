# Notes App

Simple notes app created with React.

## Preview

![preview](./public/preview.png)

## Prerequisites

- HTML
- CSS
- JavaScript
- React
- Git and Github

## Features 

- Create new notes
- Delete notes
- Search notes
- Limit the amount of characters
